## 📁 Projects Included

### 1️⃣ Multiple Dice Rolling Game 🎲

**Description:**  
A JavaScript-based dice rolling game where the user can roll multiple dice at once. Each dice roll generates a random number between 1 and 6 and displays the corresponding dice image.

**Features:**
- User selects the number of dice
- Random dice values generated using JavaScript
- Dice results displayed both as numbers and images
- Images dynamically updated using DOM manipulation
- Input validation to prevent invalid values

**Concepts Used:**
- Arrays
- Loops
- `Math.random()`
- DOM manipulation
- Dynamic image rendering
- Basic CSS styling

---

### 2️⃣ Temperature Converter 🌡️

**Description:**  
A temperature conversion tool that converts values between **Celsius and Fahrenheit** based on user selection.

**Features:**
- User input for temperature
- Radio buttons to select conversion type
- Converts Celsius → Fahrenheit and Fahrenheit → Celsius
- Input validation for empty or invalid values
- Result displayed dynamically on the page

**Concepts Used:**
- Functions
- Conditional logic
- DOM input handling
- Form controls (radio buttons)
- String interpolation
- User feedback and validation

---

### 3️⃣ Calculator Application 🧮

**Description:**
A JavaScript-based calculator that performs basic arithmetic operations with proper input handling and undo functionality.
The project focuses on understanding UI state, user input control, and dynamic updates using JavaScript.

**Features:**

Number and operator buttons for calculations

Supports addition, subtraction, multiplication, and division

Prevents invalid operator sequences

Clear (C) button to reset the calculator

Undo (backspace-style) functionality using history tracking

Result evaluation with error handling

Display updates dynamically based on user actions

4️⃣ Pokémon Generator 🔍⚡

Description:
A JavaScript web application that fetches Pokémon data from the PokéAPI and dynamically displays the Pokémon’s official artwork based on user input. This project introduces API integration and asynchronous JavaScript into my learning journey.

Features:

User enters the name of a Pokémon

Data fetched dynamically from the PokéAPI

Displays high-resolution official Pokémon artwork

Uses async/await for handling API requests

Updates the UI dynamically after fetching data

Converts user input to API-compatible format

Concepts Used:

Fetch API

Asynchronous JavaScript (async / await)

JSON data handling

DOM manipulation

Event listeners

Dynamic image rendering

API integration
